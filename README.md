# Calorie / Weight Loss Calculator

A modern, single-page calculator that estimates a daily calorie target to reach a goal
weight in a chosen number of days. It uses the Mifflin-St Jeor equation with activity
level adjustments and supports U.S. weight units (lbs) with height in centimeters.

## Features

- Activity-based TDEE estimate (low / medium / high)
- Goal-driven calorie target with daily deficit math
- Custom-styled inputs, dropdown, and stepper controls
- Clean, modern UI with a glassy aesthetic

## Tech Stack

- HTML
- Tailwind CSS (via CDN)
- Vanilla JavaScript

## Project Structure

- `index.html` - markup and layout
- `css/main.css` - styling and UI polish
- `js/index.js` - calculator logic and interactions

## Usage

Open `index.html` in a browser, enter your details, choose an activity level, and hit
Calculate to see your recommended daily calorie maximum.

## Notes

This calculator provides an estimate only. Consult a healthcare professional for
personal advice.
